    Bug Fixes

    • Fix various issues with build mode

    • Fix civilians not being revealed
    • Fix auto balance button when no faction is selected
    • Fix hot build not being reset after a build order
    • Fix beam weapons continuously restarting as they ground fire (This also fixes the bug where beam weapons that are firing on the ground apply the damage twice)

    • Fix units receiving full vet after being transferred

    • Fix base AI logic running with AIs from the vault

    Navigational mesh

    The navigational mesh is an abstract representation of a map. The AI can use that representation to gain an understanding of what the map 'looks like'. The navigational mesh is generated at the start of the map and works on any map.

    • Fix labels being culled even though they have resources
    • Introduce a search threshold when searching for a valid leaf
    • Fix navigational mesh being able to be generated multiple times
    • Fix order of execution so that expansion markers are properly generated

    Physics Based Rendering (PBR)

    We're slowly but surely preparing the game for a significant visual improvement. This is a long term project that is going to consume hundreds of hours. We're hoping to find additional contributors that have an interesting in creating (and fixing) meshes and textures. If you have that interest and/or experience and you'd like to help out then you can introduce yourself in the dedicated channels in the official Discord channel!
